# Springboot+Thymeleaf实现个人博客网站

#### 介绍
简单的使用Springboot+Thymeleaf实现个人博客网站。

#### 技术使用
前端使用Thrmeleaf模板 + SemanticUi，后端使用SpringBoot + Mybatis/Mybatis-plus。
